# 📄 CV Builder

A professional, browser-based CV builder with multiple templates, instant preview, and AI-powered CV analysis — no backend required.

**🌐 Live Demo → [metehanulusoy.github.io/cv-builder](https://metehanulusoy.github.io/cv-builder)**

---

## ✨ Features

- **4 Professional Templates** — Modern, Classic, Minimal, Creative
- **Fullscreen CV Preview** — Generate and see your CV take over the entire screen
- **Download as PDF** — Print-ready output via browser print dialog
- **📊 CV Analysis Panel**
  - CV Score (0–100) with completeness breakdown
  - Job role recommendations based on your skills & experience
  - ATS keyword suggestions to pass applicant tracking systems
  - Missing fields detector
  - Improvement tips

## 🚀 How to Use

1. Fill in your details (name, experience, skills, etc.)
2. Pick a template
3. Click **Generate CV**
4. View your CV fullscreen — download or analyze it

## 🛠️ Tech Stack

- Pure HTML + CSS + JavaScript (zero dependencies)
- No build step, no backend, no API calls
- Hosted on GitHub Pages
